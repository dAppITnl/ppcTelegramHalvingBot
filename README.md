## Setup and Installation

To set up and run this project, follow these steps:

1. **Create a Virtual Environment**:
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  pip3 install requests
  pip3 install python-telegram-bot
  pip3 list
```
